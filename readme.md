# 🎫 SGST — Sistema de Gestão de Suporte Técnico

Protótipo navegável de um sistema para registro e acompanhamento 
de chamados de suporte técnico interno em empresas de médio porte.

## 🖥️ Como acessar

👉 **https://delaneza.github.io/sgst-prototipo/**

Basta abrir no navegador, sem instalação.

## 👥 Perfis disponíveis

Na tela de login, selecione o perfil desejado:

| Perfil | Acesso |
|--------|--------|
| 👤 Funcionário | Abrir chamados e acompanhar status |
| 🔧 Técnico | Gerenciar fila e registrar soluções |
| 📊 Gestor | Painel de indicadores e relatórios |

## 📱 Telas

- **Login** — Autenticação com seleção de perfil
- **Dashboard** — Visão geral dos chamados
- **Abertura de Chamado** — Formulário com protocolo gerado automaticamente
- **Fila do Técnico** — Gerenciamento por prioridade
- **Painel do Gestor** — Indicadores e desempenho da equipe

## 🛠️ Tecnologias

HTML5, CSS3 e JavaScript — hospedado via GitHub Pages
